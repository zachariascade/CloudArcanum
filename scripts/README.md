# Scripts

Scripts are grouped around Cloud Arcanum responsibilities.

- `dev.ts`, `dev-api.ts`, and `dev-web.ts` start the local Arcanum stack
- `ingest-*`, `export-*`, `validate.ts`, and `benchmark-api.ts` support content workflows

Preferred command namespaces:

- `dev*`
- `arcanum:*`
- `ingest:*`
- `export:*`
